 #### 字符串

 | \oyy | 八进制数，yy代表的字符，例如：\o12代表换行   |
 | :------------- | :------------- |
 | \xyy       | 十六进制数，yy代表的字符，例如：\x0a代表换行 |

#####   翻转字符串
```
>> a = "abc"
'abc'
>> a[::-1]
'cba'
>>
```
####  基本操作

1. len()：求序列长度
2. + ：连接2个序列
3. * : 重复序列元素
4. in :判断元素是否存在于序列中
5. max() :返回最大值
6. min() :返回最小值
7. cmp(str1,str2) :比较2个序列值是否相同
